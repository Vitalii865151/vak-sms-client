# VAK SMS Client

Простой Python-клиент для работы с API VAK SMS.  
Позволяет получать номера, ожидать SMS-коды и отображать результаты через GUI (Tkinter).
Коды стран и сервисов: https://vak-sms.com/api/vak

## ⚙️ Функционал

- Получение номера телефона из VAK SMS
- Ожидание SMS-кода (таймаут до 4 минут)
- GUI с отображением номера и кода
- Копирование в буфер кликом по элементу
- Настраиваемые параметры: Service, Country, Operator
